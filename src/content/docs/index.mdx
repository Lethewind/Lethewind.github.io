## 开始更新

- 写新文章：在 `src/content/docs/blog` 下新增 `.md` 或 `.mdx` 文件。
- 写短笔记：在 `src/content/docs/notes` 下新增文档。
- 修改导航：编辑 `astro.config.mjs` 里的 `sidebar` 配置。
- 发布内容：`git push` 后会自动触发 GitHub Pages 部署。
